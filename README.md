# Панель администратора ипотечного калькулятора

Прототип интерфейса для управления банками-партнерами и ипотечными программами. Приложение позволяет добавлять новые банки, редактировать условия по ипотечным программам (ставки, взносы, сроки), а также управлять видимостью банков в публичном калькуляторе.

## Функциональность

- **Список банков**: Просмотр всех добавленных банков с логотипами и статусами.
- **Фильтрация**: Быстрый поиск по конкретному банку, статусу активности или наличию программ.
- **Редактирование**: Управление общей информацией о банке (название, логотип, описание).
- **Ипотечные программы**: Добавление и настройка программ (Семейная, IT, Стандартная и др.) с указанием ставок, ПВ и сроков.
- **Автоставки**: Возможность переключения программ в режим автоматического обновления ставок (симуляция).
- **Массовые действия**: Переключатель "Применить ко всем проектам".

## Технологический стек

- **React 18**
- **TypeScript**
- **Vite**
- **Tailwind CSS**
- **Lucide React** (иконки)

## Установка и запуск

1. Клонируйте репозиторий:
   ```bash
   git clone <repository-url>
   ```

2. Установите зависимости:
   ```bash
   npm install
   ```

3. Запустите режим разработки:
   ```bash
   npm run dev
   ```

4. Сборка для продакшена:
   ```bash
   npm run build
   ```
